Berikut versi **4 tabel** yang tetap sederhana, rapi, dan cocok untuk web manajemen sekolah sepakbola.

---

# 1. **Tabel Group**

Kelompok latihan (U10, U12, U14).

Kolom:

* id
* name
* coach_id (opsional, menghubungkan ke tabel Coach)

---

# 2. **Tabel Player**

Data pemain.

Kolom:

* id
* name
* age
* position
* photo
* group_id → FK ke Group

---

# 3. **Tabel TrainingSchedule**

Jadwal latihan.

Kolom:

* id
* date
* time
* group_id → FK ke Group

---

# 4. **Tabel Coach**

Data pelatih.

Kolom:

* id
* name
* license_level (opsional)
* phone
* photo

---

# Relasi antar tabel

```
Coach (1) ────< Group (many)

Group (1) ────< Player (many)

Group (1) ────< TrainingSchedule (many)
```

---

# Penjelasan

* **Coach → Group:**
  Satu pelatih bisa menangani banyak kelompok.
* **Group → Player:**
  Satu kelompok punya banyak pemain.
* **Group → TrainingSchedule:**
  Satu kelompok punya banyak jadwal latihan.
